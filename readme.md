# [Coding Compendium] - Portfolio

Een persoonlijke portfolio landing page gebouwd als afsluiting van Fase 1
van de frontend development cursus.

## Live demo

[ Bekijk de live pagina ](https://code-compendium.github.io/Fase-1-Eindopdracht/)

## Technische verantwoording

### HTML

- Semantische structuur met landmarks en correcte kopenstructuur
- Toegankelijk formulier met gekoppelde labels
- Skip link voor toetsenbordnavigatie

### CSS

- Mobile first met min-width media queries
- CSS Grid voor de paginalayout, Flexbox voor componenten
- BEM naamgeving op alle componenten
- Custom properties voor kleuren, spacing en typografie
- clamp() voor vloeiende typografie

### Toegankelijkheid

- WCAG AA kleurcontrast op alle tekst
- Zichtbare focus-stijlen op alle interactieve elementen
- prefers-reduced-motion voor geanimeerde elementen
- prefers-color-scheme voor donkere modus

### Animaties

- @keyframes fade-in op de hero sectie bij laden
- Hover transitions op kaarten en knoppen

## Gebouwd met

- HTML5
- CSS3
- Git & GitHub
- GitHub Pages
